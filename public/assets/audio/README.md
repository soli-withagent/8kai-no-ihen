# 効果音（実録音サンプル）の入れ方

このフォルダに以下のファイル名で音声を置くと、ゲームが**自動で実録音を採用**します。
無い音は高品質な合成音にフォールバックします（畳み込みリバーブでかご内の反響つき）。

| ファイル名      | 用途                         |
|-----------------|------------------------------|
| `hum.mp3`       | かご内の環境音（ループ再生） |
| `ding.mp3`      | 到着チャイム（正常）         |
| `ding_bad.mp3`  | 到着時の異音（異変フロア用） |
| `door.mp3`      | 扉の開閉音                   |
| `motor.mp3`     | 移動中のモーター音           |
| `error.mp3`     | 誤答時のエラー音             |

- 形式は `.mp3`（44.1kHz 推奨）。ファイル名は完全一致。
- 置いたら再読み込みするだけで反映されます（ビルド不要）。
- ライセンスは **CC0 / パブリックドメイン / 商用可のロイヤリティフリー** を推奨。
  入手先の例: freesound.org（CC0で絞り込み）, Pixabay, Mixkit など。
