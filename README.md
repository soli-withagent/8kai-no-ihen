# 8階の異変

「8番出口」風の一人称ホラー間違い探し。エレベーターに乗り、扉が開くたびに**かごの中の異変**を見抜きながら最上階・8階を目指す。

🎮 **プレイ:** https://8kai-no-ihen.vercel.app

## 遊び方
- 扉が開いたら、ドラッグ／矢印キー・WASD で**かごの中を見回す**
- **異変がなければ「開（招き入れる）」** — 来訪者を乗せて次の階へ
- **異変があれば「閉（逃げる）」** — 締め出して次の階へ
- 判断を誤れば…**異変に呑まれる／住人を見殺しに**。一度のミスで終わり
- 異変は全20種・約50%で出現。目指すは8階。

## 開発
- Vite + Three.js（WebGL）
- `npm install` → `npm run dev`（開発） / `npm run build`（本番ビルド）
